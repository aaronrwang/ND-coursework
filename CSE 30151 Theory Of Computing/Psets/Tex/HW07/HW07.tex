    \begin{itemize}
        \item A 10-compliant Turing machine always halts on every input
        \item It is decidable whether a Turing machine is 10-compliant.
    \end{itemize}
    In this problem, weâ€™ll show that any such set of rules will be incomplete in the sense that there is a language $L_2$ that is decidable, yet no TM that decides $L_2$ complies with the rules.\newp
